# NanoFarm

This is a mini project which is farm application, using node.js file system.

# Techonologies Used:

- html5
- css3
- Nodejs
- Expressjs
- File System

# Key Features:

Some of the standout features of the project are:

1. Multiple choices of farm items
2. Check out option
3. Vegetable Details
